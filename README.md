# Coming Soon Countdown Website

A modern and professional "Coming Soon" webpage with a live countdown timer, designed using HTML, CSS, and JavaScript. The page features smooth animations, a clean layout, and a fully responsive design suitable for all screen sizes.

## Features
- Live countdown timer showing days, hours, minutes & seconds
- Fully responsive layout for mobile, tablet, and desktop
- Clean and visually appealing UI
- Smooth animations and transitions
- Launch message auto-update after countdown completion (if added)

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation
Clone the repository bash  git clone https://github.com/shiprasonal/Coming-Soon-Countdown-Website.git  
Then open `index.html` in your web browser.

## Usage
Open the website in any browser to view the countdown timer.  
Can be used for upcoming website/product launches, event countdowns, and promotional pages.
